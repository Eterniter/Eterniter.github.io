Hugo添加Algolia搜索

发表于2022-07-10|hugo
|字数总计:99|阅读时长:1分钟|阅读量:|评论数:

本文介绍如何在Hugo LoveIt主题中添加Algolia搜索

1. 在algolia官网注册登录之后，创建应用

2. 选择区域，确认应用创建完成，然后创建索引

3. 运行hugo命令，生成public文件夹

4. 上传index.json

5. 进入API页面

6. 配置config.toml

将之前的索引名、Application ID、Search-Only API Key依次填入params.search.algolia即可。
